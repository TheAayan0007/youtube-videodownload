How It Works

Up and running
in 60 seconds.

Four steps from zero to downloading your first video at 4K.

01
Run the Installer
Execute python install.py. It checks your Python version, installs PyQt6, yt-dlp, Pillow, and downloads FFmpeg for your OS automatically.
02
Launch Prism
Run python prism.py. On first launch you'll see the animated PRISM splash screen before the main window appears.
03
Paste & Fetch
Paste any YouTube URL into the search bar and press Fetch. Prism extracts full video metadata, available formats, and the thumbnail in the background.
04
Choose & Download
Select quality, audio track, and format mode. Hit Download, choose your folder, and watch the real-time progress bar. Done — check your History tab.

Installation

Zero to downloading
in 60 seconds.

Prism ships with an automated installer that handles everything — Python deps, FFmpeg, the works.

🪟
Windows 10 / 11
Installer auto-downloads FFmpeg from gyan.dev and adds it to your user PATH via registry. Python 3.10+ required.
🍎
macOS (Apple Silicon + Intel)
Uses Homebrew to install FFmpeg. If Homebrew isn't present, guides you to install it first. Python 3.10+ from python.org or Homebrew.
🐧
Linux (All Major Distros)
Detects apt / dnf / pacman / zypper / apk. Falls back to johnvansickle.com static binary with ARM64 support. Works rootless via ~/.local/bin.

Tech Stack

Built on proven,
open-source tools.

🐍
Python 3.10+
Clean, modern Python. Threading, pathlib, JSON persistence, URL parsing — all stdlib. No bloat.
🖼
PyQt6
Industrial-strength Qt bindings. Custom QPainter for every visual effect, animated backgrounds, and GPU-accelerated rendering.
⬇
yt-dlp
The gold-standard YouTube downloader library. Format selection, playlist metadata, progress hooks, multi-stream merging.
🎬
FFmpeg
Merges separate video and audio streams into a single MP4. Handles codec conversion, remuxing, and ensures output compatibility.
🖼
Pillow
Fetches and converts thumbnails from remote URLs. Handles JPEG/PNG conversion, resizing, and alpha compositing for the UI.
📦
Zero Dependencies on Runtime
All non-standard libs are auto-installed at first run. No virtual environments required. Just Python + the installer.
