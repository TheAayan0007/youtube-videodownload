How It Works

Three modes.
Infinite use cases.

01
Paste any YouTube URL
Paste a full URL, short youtu.be link, or a video with timestamps. Prism normalises everything automatically.
02
Fetch video metadata
Click Fetch to pull title, channel, views, likes, duration, available formats and the highest-resolution thumbnail.
03
Choose your format & quality
Select Video+Audio, Audio Only or Video Only. Fine-tune quality from 4K to 360p and choose your language track.
04
Download & save
Click Download, pick a folder and watch real-time speed + percentage. Files are saved as MP4 via FFmpeg merging.

System Requirements

What you need
to run Prism.

The installer handles everything automatically — this is just for reference.

🐍
Python
3.10 or higher
The runtime for Prism. The installer will guide you through upgrading if your version is older.
🪟
PyQt6
Latest
The GUI framework powering the entire Prism interface. Installed automatically via pip.
📥
yt-dlp
Latest
The powerful download engine behind Prism — supports thousands of sites beyond YouTube.
🖼
Pillow
Latest
Image processing for thumbnails, format conversion (JPG/PNG) and display within the app.
🌐
requests
Latest
HTTP library used to fetch thumbnails and media data from remote URLs efficiently.
🎞
FFmpeg
5.0+
Video/audio merging and container conversion. Auto-installed for all platforms by the setup script.
