Features

Everything built in.

No plugins, no extensions, no subscriptions — just one app that does it all.

🎬
Video Download
Paste any YouTube URL. Fetches all metadata, format options and thumbnail. Download from 8K all the way down to 360p.
Core
🎵
Audio Only
Extract high-bitrate audio streams with full language track selection. Perfect for music, podcasts and lectures.
Audio

Playlist Batch
Load entire playlists, pick individual videos, choose quality per-video and batch-download with a live queue manager.
Batch

Download the highest-resolution thumbnail from any video in JPG or PNG format at full native resolution.

Every download logged with title, duration, save path and timestamp. Persists in a local JSON file — no cloud needed.
Persistent

Live speed meter, percentage and animated progress bars per video and for the overall batch. Fully non-blocking UI.
Live

Control

Personalise

Installation

Two ways
to install.

Python method works on every platform. The EXE is Windows one-click.

01
Python Method
ALL PLATFORMS
1
Clone the repository Get source code from GitHub
$git clone https://github.com/TheAayan0007/prism
2
Run the installer Auto-installs PyQt6, yt-dlp, FFmpeg and all dependencies
$python installer.py
3
Launch Prism Run after a successful installation
$python main.py
Windows
macOS
Linux
02
Executable Method
WINDOWS ONLY
1
Download installer.exe Sets up all required runtimes — run this first
2
Download Prism.exe The main app — double-click to launch and start downloading immediately
