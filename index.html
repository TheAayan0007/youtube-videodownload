⚙ BACKEND SETUP GUIDE

    

      
This web UI is a pure frontend — it builds yt-dlp commands you can run locally.

      
To enable one-click downloading, run this minimal Python backend on your machine:

      pip install yt-dlp flask flask-cors
python nexus_backend.py
      
Then open http://localhost:5000 — the UI auto-detects the backend and enables direct downloads.

      
⚠ You can also run yt-dlp directly from the generated commands in your terminal.
